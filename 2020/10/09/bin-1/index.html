1、 Write a hello world assembly program

先看数据段

定义msg db “Hello，world！”，10 定义了字符串

db指令给一块连续的存储区域装载字节型数据

len equ $ - msg 获得字符串长度

equ指令定义常数。$代表当前的偏移值，$-msg偏移相减得到字符串的长度

代码段

global _start指定了入口。

sys_write的系统调用号是4，int 80是系统调用指令，调用号存放在eax寄存器中。

ebx存放第一个参数1，文件描述符1代表stdout标准输出。

ecx存放第二个参数msg，输出的字符串。

edx存放第三个参数len，输出长度。

相当于调用函数write（1，msg，len）

最后调用exit退出程序。exit的系统调用号为1

#编译连接

nasm -f elf64 hello.asm -o hello.o（可以指定32位或者64位）

#生成可执行文件

ld -o hello hello.o（链接32位添加参数-melf_i386）

#运行文件

./hello

3、Write a procedure named sum that adds the variables stored in the ECX and EDX register and returns the sum in the EAX register.

直接看代码段

将字符‘4’和‘5’放入ecx和edx，都减去字符‘0’得到偏移

定义了一个sum函数

将ecx和edx都加到eax中，再加上‘0’的偏移，这时eax中存放的应该是字符‘9’

声明.bss段 未初始化数据

resb指令声明未初始化的字节空间

这里res resb 1声明了1字节的空间

mov [res],eax将eax的值（字符‘9’）存放在res的空间，然后就是调用write输出The sum is字符串

然后再将res中的值给ecx作为第二个参数，调用write输出字符9

最后再退出

5、Write a program allocates 16kb of memory using the sys_brk() system call.

jl指令，有符号小于则跳转。

xor指令对两个操作数进行异或运算，结果保存在目标操作数中。

xor eax ，eax相当于清零eax

std指令将方向标志位DF置1，
在字串操作中使SI或DI的地址指针自动递减，字串处理由后往前。

rep时重复字符串操作前缀。

stosd指令将eax的内容存放到edi指向的位置，同时edi的值根据方向标志的值增加或者减少。

cld指令将标志寄存器Flag的方向标志位DF清零。
